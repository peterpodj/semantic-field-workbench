## SGD Court Procedure (Domain Variant)

1. **Mini-Batch** — Sample only what the canvas interval can hold: one word token, one Gaussian source, one parameter set (D, uX, uY, k)

2. **Gradient** — Examine by the 5 title questions: Is this diffusion physical or a 5-point stencil artifact? Is this advection direction justified by the IT domain or arbitrarily chosen? Is the source injection rate lawful?

3. **Step** — Move based on partial evidence with a lawful learning rate: adjust D in increments that preserve dt < dx²/(4D); perturb u by small vectors; add sources sparingly so the field remains readable

4. **Noise** — Inject contradiction to test local minima: swap palette theme and observe if interpretation changes without numerical change; coarsen grid mentally and ask if the metaphor still holds; reverse advection direction and check boundary behavior

5. **Epoch** — Return after interval with new data (new source word, new IT domain card), reshuffle parameters, re-hold the field state. The simulation loop is the epoch. 🌊
